Reset Semua Data?
Tindakan ini akan menghapus seluruh data yang sudah diinput:
jumlah riil, harga, stok SW, expired date,
data CSV Stockwiz, info toko & tanggal.

Data tidak dapat dikembalikan setelah direset.
Mulai Stock Opname Baru?
Masih ada data input dari sesi sebelumnya yang belum direset. Untuk memulai stock opname baru, seluruh data input yang ada akan dihapus:
jumlah riil, harga, stok SW, expired date,
data CSV Stockwiz.

Unit kerja, tanggal, jam, dan auditor yang sudah diisi tetap disimpan. Data tidak dapat dikembalikan setelah direset.
